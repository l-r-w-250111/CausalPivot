# CausalPivot
Logical at its core, bold in its vision: An AI for invention.
「因果の理解」「自己進化」「創造的跳躍」を統合

- Core A: Causal Core（因果）
  - 原因候補
  - 介入可能性
  - 判別実験
  - 反事実、do介入
  - 時間発展
  - 構造検証

- Core B: Growth Core（自律成長）
  - 失敗履歴
  - 目標階層
  - phase / abstraction
  - guidance snapshot
  - principle persistence
  - growth log
  - メタ認知

- Core C: Leap Core（飛躍）

### 全体アーキテクチャ
#### 発明
 
- Input  
↓  
- Idea Phase 
  - 内部マイクロ分岐（hidden branching）
  - 複数素案生成  
↓  
- Rationalize Phase  
  - 因果・S行列による制約付与
  - 意味づけ（ノード群単位）  
↓  
- Check Phase  
  - 論理・物理・観測可能性チェック
  - PASS / FAIL / INDETERMINATE  
↓  
- Decide Phase 
  - 集約しない
  - 推奨検討順序・理由を提示  
↓  
- Human Judgment / Real Experiment
